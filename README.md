# 飲酒カウンター

個人用の飲酒量管理Webアプリ。当日の飲酒缶数・酒レベル・週間累計を記録する。

**URL**: https://ricckyyy.github.io/drink-counter/

## 機能

- ビール / ワイン / サワー・ジントニ の3種類をカウント
- タップ +1、長押し（500ms）−1
- 酒レベル表示（素面 / 爽快期 / ほろ酔い期 / 酩酊初期 / 酩酊極期 / 泥酔期）
- 週間ゲージ（目標 5缶 / 週）
- データは `localStorage` に保存（オフライン動作）

## 酒レベル定義と根拠

レベルは厚生労働省 e-ヘルスネット等に基づく医学的な酔いの6段階を採用。
1缶 = ビール350ml / アルコール度数5% = **純アルコール約14g** として換算。

| レベル | 缶数（350ml缶） | ラベル | 血中アルコール濃度（目安） | 純アルコール量 |
|---|---|---|---|---|
| Lv0 | 0缶 | 素面 | — | 0g |
| Lv1 | 1缶 | 爽快期 | 0.02〜0.04% | 〜14g |
| Lv2 | 2〜3缶 | ほろ酔い期 | 0.05〜0.10% | 28〜42g |
| Lv3 | 4〜5缶 | 酩酊初期 | 0.11〜0.15% | 56〜70g |
| Lv4 | 6〜9缶 | 酩酊極期 | 0.16〜0.30% | 84〜126g |
| Lv5 | 10缶以上 | 泥酔期 | 0.31%〜 | 140g〜 |

### 健康上の参考値

- 厚労省「節度ある飲酒」の目安：**1日 純アルコール20g（約1.4缶）**
- 生活習慣病リスクが高まる量：男性 **40g以上（約3缶〜）**、女性 **20g以上（約1.4缶〜）**
- 個人差（性別・体重・代謝能力）により同じ缶数でも酔いの程度は大きく異なる
